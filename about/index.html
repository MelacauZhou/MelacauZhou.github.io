About me

这是一个本科新生的个人网站，是在基于第一学期某门课程的半期作业上进行拓展的一个项目。

本网站基于 hexo 搭建， 采用 cactus-white 主题，在 Mac OS 系统上进行生成后上传到github上。

在完成项目作业的工程中，作者想起了曾经在高中参加信息学竞赛时利用第三方平台搭建博客作为学习记录的经历，现在打算将这个网站同样用作作者参加 ICPC 类竞赛的记录网站，网站将侧重于套题的解题报告以及一些算法的学习记录。

​ This is a personal website of an undergraduate freshman, which is an project based on a half-term assignment for a course in the first semester.

​ This website is built based on hexo with cactus-white theme and is generated on Mac OS system and uploaded to Github.

​ During accomplishing the project, the author recalls his experience of using a third-party platform to make a blog recording learning experiences when participating the informatics competitions in high school. Now, He is going to use this web site for recording the experience of ICPC contests, and the web site will focus on analysises of problem sets and records of learning algorithms.
