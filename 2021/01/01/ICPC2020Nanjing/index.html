E. Evil Coordinate

Description

无限二维平面上有一个地雷，给定机器人移动方向 UDLR 的个数，构造一个序列使得其避开地雷走到目标点。

Analysis

除去特殊情况后将一次性一直往同方向走，枚举全排列一定存在方案。时间复杂度 $O(4! \times n)$。

F. Fireworks

Description

制造一个烟花的时间为 $n$, 一次性放完所有烟花的时间为 $m$，每个烟花有 $p$ 的概率绽放，求期望最短绽放烟花的时间。

Analysis

显然每次放的烟花数量是一定的，满足单峰性，三分后计算即可。

H. Harmonious Rectangle

Description

给 $N \times M$ 的格子涂上三种颜色，求存在一个 $2 \times 2$ 的子矩形满足行颜色相同或列颜色相同。

Analysis

抽屉原理，当其中一维 $\ge 8$ 时无论如何涂色必然存在一种方案，小范围暴力计算不可行方案数（或打表），大范围直接输出方案数。

I. Interested in Skiing

Description

一定宽度的无限长的滑雪道上有着若干个矩形障碍物，给出沿滑雪道下滑的固定速度，求最小需要有多少左右移动的速度上限。

Analysis

显然只有矩形的角有用，二分速度后直接bfs check即可。

J. Just Another Game of Stones

Description

维护数据结构支持：

- 修改某一区间石子数变成 $max(v_i, x)$
- 询问某一区间的石子进行nim游戏，先手有多少种可行的第一步使得其必胜。

Analysis

先考虑询问，设区间异或和为 $x$，记其二进制最高位为 $b(x)$，那么显然我们需要找到满足 $(v_i - y)\ xor\ x\ xor \ v_i = 0, y > 0$ 的 $i$ 的个数，将左式变形合并，得到：$v_i - y = x\ xor\ v_i , y > 0$ 即 $v_i - (x \ xor \ v_i ) > 0$ ，显然当 $v_i$ 满足 $b(x)$ 位为1时，左式成立，反之左式不成立，固只需要分别对每个二进制位维护数量和即可。

在考虑修改操作，维护信息是支持 Segment Tree Beats! 的，固采用该方法即可解决。 时间复杂度 $O((q\log{2}n+n)\log{2}v)$。

Code

K. K Co-prime Permutation

Desciption

构造一个 $1\sim n$ 的排列 $p$ 满足恰好有 $k$ 组数满足 $gcd(p_i, i) = 1$。

Analysis

对 $1 \sim k$ 做一次轮换即可。

Code

L. Let’s Play Curling

Description

给定两个长度为 $n$ 的数组，确定一个中心 $c$ 使得 $a$ 数组中满足比 $b$ 中任意元素离 $c$ 都更近的元素尽可能的多的答案。

Analysis

本质上求排序后 $b$ 的每个相邻区间内有多少个 $a$ 元素。

Code

M. Monster Hunter

Description

给定一棵 $n$ 个点的带权有根树，对于 $m = 0 \sim n$ 求任意 $m$ 个点去掉之后，$\sum hp{i}+\sum{i\ isn’t\ root}hp_i$的最小值。

Analysis

$dp[i][j][0/1]$ 表示以 $i$ 为根的子树内，删去了 $j$ 个点，$i$ 号点是否被删去的答案，转移比较简单，利用 dsu on tree 可以做到 $O(n^2)$。
