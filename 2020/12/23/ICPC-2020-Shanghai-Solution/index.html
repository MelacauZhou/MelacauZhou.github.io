B. Mine Sweeper II

Description

​ 给定两个 \(n \times m\) 的黑白网格图 $ A, B $，求一种将 \(B\) 中不超过 \(\lfloor \frac{nm}{2}\rfloor\) 的格子翻转颜色的方案使得黑白联通点对数与 \(A\) 的点对数相同。

​ \(n, m \leq 1000\)，输出翻转后的网格图。

Analysis

​ 容易发现，将黑白全部翻转之后的点对数与翻转前相同，因此输出 \({A, rev(A)}\) 中与 \(B\) 差异较少的图输出即可。

C. Sum of Log

Description

\(T\) 组数据，求 \(\sum_{i=0}^X\sum_{j=[i==0]}^Y[i \And j = 0](\lfloor log_{2}{(i + j)}\rfloor + 1)\) , \(T \leq 10^4, X, Y \leq 10^9\)

Analysis

​ 数位DP，设 \(dp[k][0/1][0/1]\) 表示大于 \(k\) 的二进制位已经合法，\(X,Y\) 在第 \(k\) 位分别是什么的答案。

​ 那么，\([i\And j=0]\) 限制了 \(i\) 与 \(j\) 不相交，同时也保证了 \(\lfloor \log_{2} (i +j)\rfloor = \lfloor\log_{2} \max\{i,j\} \rfloor\), 因此我们可以很容易的得出几种转移，注意卡 \(X\) 和 \(Y\) 上限时的操作即可。

​ 时间复杂度： \(O(30k \times T), k\) 为转移数​。

D. Walker

Description

​ 有两把刷子，进行粉刷的速度分别为 \(vx, vy\) ，同时有一个长度为 \(n\) 的线段，给定两把刷子的初始位置，求最快什么时候能将线段完全覆盖。

Analysis

​ 分类讨论之后一共大致有三种情况：一个人直接刷完，两个人往对方那边走，两个人各自负责一段，注意一些边界情况以后暴力解方程计算即可。时间复杂度。\(O(kT)\)，\(k\) 为情况数。

E. The Joueney of Geor Autumn

Description

有一个 \(1\sim n\) 的排列，给定 \(k\) ，求满足 \(\forall k < i \leq n, p_{i}>\min_{j=1}^kp_{i-j}\)的排列数。\(n, k \leq 10^7\)

Analysis

考虑 DP，定义 \(f_{i}\) 表示前 \(i-1\) 个数已经合法，且第 \(i\) 个数填的是后 \(n - i + 1\) 个数中的最小值的情况，那么转移为 \(f_{i} = \sum_{j=1}^{k}f_{i-j}A_{n-i+j-1}^{j-1}\) 将其转化为阶乘展开后，发现转移时与 \(j\) 无关，做前缀和后可以 \(O(1)\) 转移。
