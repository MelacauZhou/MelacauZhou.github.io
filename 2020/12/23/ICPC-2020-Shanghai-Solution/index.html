B. Mine Sweeper II

Description

​ 给定两个 \(n \times m\) 的黑白网格图 $ A, B $，求一种将 \(B\) 中不超过 \(\lfloor \frac{nm}{2}\rfloor\) 的格子翻转颜色的方案使得黑白联通点对数与 \(A\) 的点对数相同。

​ \(n, m \leq 1000\)，输出翻转后的网格图。

Analysis

​ 容易发现，将黑白全部翻转之后的点对数与翻转前相同，因此将 \({A, rev(A)}\) 中与 \(B\) 差异较少的图输出即可。

C. Sum of Log

Description

\(T\) 组数据，求 \(\sum_{i=0}^X\sum_{j=[i==0]}^Y[i \And j = 0](\lfloor log_{2}{(i + j)}\rfloor + 1)\) , \(T \leq 10^4, X, Y \leq 10^9\)

Analysis

​ 数位DP，设 \(dp[k][0/1][0/1]\) 表示大于 \(k\) 的二进制位已经合法，\(X,Y\) 在第 \(k\) 位分别是什么的答案。

​ 那么，\([i\And j=0]\) 限制了 \(i\) 与 \(j\) 不相交，同时也保证了 \(\lfloor \log_{2} (i +j)\rfloor = \lfloor\log_{2} \max\{i,j\} \rfloor\), 因此我们可以很容易的得出几种转移，注意卡 \(X\) 和 \(Y\) 上限时的操作即可。

​ 时间复杂度： \(O(30k \times T), k\) 为转移数​。

Code

D. Walker

Description

​ 有两把刷子，进行粉刷的速度分别为 \(vx, vy\) ，同时有一个长度为 \(n\) 的线段，给定两把刷子的初始位置，求最快什么时候能将线段完全覆盖。

Analysis

​ 分类讨论之后一共大致有三种情况：一个人直接刷完，两个人往对方那边走，两个人各自负责一段，注意一些边界情况以后暴力解方程计算即可。时间复杂度。\(O(kT)\)，\(k\) 为情况数。

Code

E. The Joueney of Geor Autumn

Description

有一个 \(1\sim n\) 的排列，给定 \(k\) ，求满足 \(\forall k < i \leq n, p_{i}>\min_{j=1}^kp_{i-j}\)的排列数。\(n, k \leq 10^7\)

Analysis

考虑 DP，定义 \(f_{i}\) 表示前 \(i-1\) 个数已经合法，且第 \(i\) 个数填的是后 \(n - i + 1\) 个数中的最小值的情况，那么转移为 \(f_{i} = \sum_{j=1}^{k}f_{i-j}A_{n-i+j-1}^{j-1}\) 将其转化为阶乘展开后，发现转移时与 \(j\) 无关，做前缀和后可以 \(O(1)\) 转移。

G. Fibonacci

Description

​ 求 \(\sum_{i=1}^n\sum_{j=i+1}^n[fib_i\times fib_j\equiv0\pmod2]\)。

Analysis

​ Fibonacci 对2取余的循环节为 1,1,0 直接计算即可。

H. Rice Arrangement

Description

有一个长度为 \(N\) 的圆桌，有 \(K\) 个人在座位上，还有 \(K\) 道菜，要一一匹配菜和人，求桌子最小旋转的总弧度。

\(K \leq N \leq 10^9, K \leq 1000, \sum K \leq 5 \times 10^3\)

Analysis

由于 \(K\) 很小，排完序之后将饭和菜直接一一错位匹配即可。

时间复杂度 \(O(k^2 \log_{2}k)\)

I. Sky Garden

Description

给定 \(N\) 个同心圆，第 \(i\) 个圆的半径为 \(i\)，将这些圆等分成 \(k\) 份，求所有交点两两之间的最短距离之和。

Analysis

对于每个圆，由于对称性，只需要考虑其中一个点即可，加上一些单调性优化可以做到 \(O(n^3)\) 甚至更快。

K. Traveling Merchant

Desciption

一张无向图，点分黑白色，离开一个点后点本身会变色，求是否有无限路径。

Analysis

考虑对异色边建图以起点为根建dfs树，考虑以同色边相连的两个点，如果是返祖边，一定是合法的；如果是横叉边，那么必然需要满足其中一个点与其lca在同一个点双内且lca不是割点。可以证明只有这两种构造方法是合法的。时空复杂度 \(O(n\log n)\)

L. Traveling in the Grid World

Description

给定 \(n \times m\) 的格点图，两点之间有路径当且仅当两点之间的路径不经过整点。求从左下角走到右上角的最短欧几里得距离。

Analysis

可以分析出来，两点之间的路径 \((dx, dy)\) 合法，当且仅当 \(gcd(dx, dy) = 1\)。可以证明，最终路径最多只会由两条路径构成，直接枚举+check即可。时间复杂度 \(O(N)\)。

M.Gitignore

Description

Your git project (you don't need to be familiar with git to solve this problem) has some files that should be ignored from synchronizing. You need to calculate the minimum number of lines needed for gitignore.

Formally, your project is a folder. A folder can have files and sub folders. There are no empty folders (i.e. folders without any files or sub folders inside). Initially, the git software will synchronize all the files in your project. However, you can specify some files and folders in the settings (which is called gitignore) to exclude them from synchronizing. For each line in gitignore, you can specify either a file or all the files in a folder. You can not ignore the whole project folder (i.e. an empty line in gitignore).

You are given paths for all the files in the project and whether they should be ignored or shouldn't. Your task is to calculate the minimum number of lines for gitignore.
